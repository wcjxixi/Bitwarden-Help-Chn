# 更改存在风险的密码

{% hint style="success" %}
对应的[官方文档地址](https://bitwarden.com/help/change-at-risk-passwords/)
{% endhint %}

如果您收到主题为 **{Organization} has identified at-risk passwords**（{Organization} 已识别出存在风险的密码）的电子邮件，或者在浏览器扩展中看到**审查并更改存在风险的密码**横幅，则表示您的管理团队已识别出弱密码、重复使用的密码或已暴露的密码，并要求您更改这些密码：

{% embed url="https://bitwarden.com/assets/2N1WqXBUVvutN9qQR73WbQ/ad56aeac00234010b56e1cc74a4ba542/2025-04-25_13-08-23.png?w=980&fm=avif" %}
更改存在风险的密码
{% endembed %}

在浏览器扩展中，点击横幅，然后使用每个存在风险的密码旁边的**更改**按钮，将弱密码、重复使用的密码或已暴露的密码替换为强密码。多数情况下，浏览器扩展会将您直接引导至该服务的「更改密码」页面。请注意：

* 您必须在网站的「更改密码」表单中以及 Bitwarden 中同时更改此密码。
* 建议使用[密码生成器](../your-vault/security-tools/generator.md)创建强且唯一的新密码。
