## 更改 App 语言 <a href="#change-app-language" id="change-app-language"></a>

要更改 Bitwarden App 的语言：

{% tabs %}
{% tab title="网页 App" %}
Bitwarden 网页 App 默认使用您的网页浏览器语言，而网页浏览器语言通常由您设备的系统语言决定。要手动覆盖此默认设置并为 Bitwarden 网页 App 设置其他语言：

1、从导航中选择**设置** → **外观**：

<div data-with-frame="true"><figure><img src="https://bitwarden.com/assets/7vKmhsOfJqieQbYRxALV75/0ed7782f100100ab3d86567e6b1b0a9d/2025-12-31_10-27-36.png?w=1044&#x26;fm=avif" alt=""><figcaption><p>网页 App 外观</p></figcaption></figure></div>

2、从**语言**下拉菜单中选择一种语言。
{% endtab %}

{% tab title="桌面端" %}
Bitwarden 桌面 App 默认使用您设备的系统语言，要覆盖此默认设置并为 Bitwarden 桌面 App 设置其他语言：

1、打开桌面 App 的**偏好设置**面板（在 Windows 上，**文件** → **设置**）（在 macOS 上，**Bitwarden** → **偏好设置**）。

2、滚动到 **App 设置**部分的底部，然后使用**语言**下拉菜单选择您的语言。

<div data-with-frame="true"><figure><img src="https://bitwarden.com/assets/6NSZV46oKRi7Roeu7g2ATZ/8a7cacb71d48d47bb11c7ef46ec88903/2026-04-23_11-12-59.png?w=800&#x26;fm=avif" alt=""><figcaption><p>桌面 App 语言设置</p></figcaption></figure></div>
{% endtab %}

{% tab title="移动端" %}
Bitwarden 移动 App 默认使用您设备的系统语言，要覆盖此默认设置并为 Bitwarden 移动 App 设置其他语言：

1、打开移动 App 然后点击**设置**选项卡。

2、点击**外观**。

3、定位到**语言**字段然后选择您的首选语言。

<div align="left" data-with-frame="true"><figure><img src="https://bitwarden.com/assets/5pqOwt6W99sLHRCpN1DX1w/bc3581ebc015c0343c116c5eb609e2f5/2025-01-21_15-19-31.png?w=715&#x26;fm=avif" alt=""><figcaption><p>移动端语言设置</p></figcaption></figure></div>

4、重新启动应用程序以应用更改。
{% endtab %}

{% tab title="浏览器扩展" %}
Bitwarden 浏览器扩展默认使用您的网页浏览器语言，而网页浏览器语言通常由您设备的系统语言决定。浏览器扩展本身没有更改语言的设置，更改网页浏览器的语言也会更改扩展的语言。

* [了解如何更改 Chrome 的语言](https://support.google.com/chrome/answer/173424?co=GENIE.Platform%3DDesktop\&hl=zh-Hans)
* [了解如何更改 Firefox 的语言](https://support.mozilla.org/zh-CN/kb/%E4%BD%BF%E7%94%A8%E8%AF%AD%E8%A8%80%E5%8C%85%E6%94%B9%E5%8F%98Firefox%E7%95%8C%E9%9D%A2%E8%AF%AD%E8%A8%80)
{% endtab %}
{% endtabs %}

## 看不到您的语言吗？ <a href="#dont-see-your-language" id="dont-see-your-language"></a>

如果您的语言没有在网页密码库或桌面 App 中列出，或者您的浏览器扩展或移动 App 没有动态地使用您的语言，**我们需要您的帮助！**

Bitwarden 使用一个名为 [Crowdin](https://crowdin.com/) 的翻译工具来管理我们在许多不同语言上的本地化工作（**无需编程知识**）。

* 要对现有的翻译做出贡献或进行修改，[请加入我们的项目](https://crowdin.com/projects/kspearrin)。
* 要开始将 Bitwarden 翻译成一种新的语言，请加入我们的项目并联系[项目所有者](https://crowdin.com/profile/tgreer)。
