# 更改存在风险的密码

{% hint style="success" %}
对应的[官方文档地址](https://bitwarden.com/help/change-at-risk-passwords/)
{% endhint %}

如果您在浏览器扩展中看到一个**审查并更改存在风险的密码**或**您的密码存在风险，请立即更新**的提示，则表示您的密码库中存在需要替换为更强密码的密码。出现此横幅的原因有两种：

* 您所在组织的管理团队已识别出弱密码、重复使用的密码或已暴露的密码，并要求您更新该密码。在这种情况下，您可能还会收到组织发送的电子邮件，提醒您更新凭据。
* 作为高级用户，底层的[密码库健康报告](vault-health-reports.md)已自动识别出弱密码、重复使用的密码或已暴露的密码，Bitwarden 建议您将其作为提升安全防护的重要契机。

更改存在风险的密码
{% endembed %}

在浏览器扩展中，点击此提示，然后使用每个存在风险密码旁边的**更改**按钮，以将弱密码、重复使用的密码或已暴露的密码替换为强密码。大多数情况下，浏览器扩展会将您直接引导至该服务的「更改密码」页面。请记住：

* 您必须在网站的「更改密码」表单中以及在 Bitwarden 中同时更改此密码。
* 您应该使用[密码生成器](generator.md#liu-lan-qi-kuo-zhan)创建一个强且唯一的新密码。
