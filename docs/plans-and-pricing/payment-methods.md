# 付款方式

{% hint style="success" %}
对应的[官方文档地址](https://bitwarden.com/help/payment-methods/)
{% endhint %}

Bitwarden 接受付费方案的多种付款方式。付款方式因您的方案和计费国家/地区而异。

## 可接受的付款方式 <a href="#accepted-payment-methods" id="accepted-payment-methods"></a>

当您在网页 App 中升级或[更新计费信息](update-your-billing-information.md)时，将列出所有可用的付款方式。已存档的付款方式将用于自动[高级版方案续费](password-manager/premium-renewal.md)和[组织方案续费](organization-renewal.md)，以确保您不会意外失去对付费功能的访问权限。当您添加[附件](../password-manager/your-vault/vault-items/file-attachments.md#add-storage-space)存储空间时，也会使用此付款方式。

### 信用卡或借记卡 <a href="#credit-or-debit-card" id="credit-or-debit-card"></a>

所有 Bitwarden 方案类型都接受主流的信用卡，包括 Visa、Mastercard、American Express 和 Discover。卡片付款通过 Stripe 安全处理。在首次注册或[更新您的计费信息](update-your-billing-information.md)时，卡片详细信息将被添加到网页 App 中。

### PayPal <a href="#paypal" id="paypal"></a>

PayPal 可用作所有方案的付款方式。您可以通过绑定 PayPal 账户进行自动付款或添加[账户余额](payment-methods.md#account-credit)。PayPal 包含内置的买家保护和其他安全功能。要使用 PayPal，请在结账时或更新您的计费信息时选择此方式。

### 银行账户 <a href="#bank-account" id="bank-account"></a>

银行账户付款可用于某些团队版和企业版方案：

* 在美国境内进行国内转账时，您可以通过[付款详细信息](update-your-billing-information.md)或通过账单计费设置直接银行存款 (ACH)。ACH 付款使用 Stripe 的银行转账系统处理。
* 从美国境外的电汇只接受[账单计费](payment-methods.md#invoiced-billing)。出具账单后，您将在账单上找到我们的银行账户详细信息以完成电汇。要确认您的账户有资格进行国际电汇付款并切换到账单计费，请[联系我们](https://bitwarden.com/contact/)。

### 支票 <a href="#check" id="check"></a>

在您通过账单付款时，年按的团队版和企业版订阅可接受公司支票。邮寄地址列在您的账单上。为确保快速处理，支票上必须包含您的公司名称、账单号码和联系信息。

### 加密货币 <a href="#cryptocurrency" id="cryptocurrency"></a>

接受比特币等加密货币。您需要购买[账户余额](payment-methods.md#account-credit)才能用加密货币来升级或续订您的订阅。

### 账户余额 <a href="#account-credit" id="account-credit"></a>

账户余额允许您提前支付您的 Bitwarden 升级或订阅费用。通过 PayPal 使用当地货币或通过 BitPay 使用加密货币购买余额。如果您想通过账户余额支付，请确保在升级或您的续订日期之前购买账户余额。我们将从您的账户余额中扣除升级或续订费用。

要为您的账户添加余额：

1. 前往**设置** → **订阅**。
2. 选择**添加余额**。
3. 输入您希望购买的金额（美元）。
4. 选择**提交**。
5. 您将被跳转至 PayPal 或 BitPay 来完成交易。交易完成后，您将收到一封确认电子邮件。

{% hint style="info" %}
要使用账户余额支付未付款的账单，请[联系我们](https://bitwarden.com/contact/)。
{% endhint %}

## 账单计费 <a href="#invoiced-billing" id="invoiced-billing"></a>

在您升级或续费当天，大多数付款方式会自动处理。某些方案允许您选择通过电子邮件支付账单。使用此选项时，账单将在您的续费当天发送给计费联系人。使用账单上列出的任一方式手动支付此账单。

{% hint style="info" %}
请确保您的计费联系人的电子邮箱地址是最新的，以避免错过账单。对于个人高级版方案，这是您的账户电子邮箱。对于组织，只有所有者可以查看和更新他们的计费联系人。
{% endhint %}

谁有资格使用账单计费，以及账单付款方式何时到期，取决于：

| 订阅类型                                                           | 付款方式到期日期    |
| -------------------------------------------------------------- | ----------- |
| 按年的团队版和企业版订阅在[联系我们](https://bitwarden.com/contact/)后切换到账单计费    | 自账单日期起 30 天 |
| 通过 [Bitwarden 销售](https://bitwarden.com/contact-sales/)设置的年度方案 | 自账单日期起 30 天 |
| [经销商](bitwarden-resellers.md)                                  | 自账单日期起 45 天 |

{% hint style="info" %}
如果您使用的是其他方案且我们无法处理您的自动付款，您的**账单历史记录**中将出现[未支付的账单](invoices-and-receipts.md)。为避免服务中断，您需要在账单日期起 14 天内付款。如果您需要帮助，请[联系我们](https://bitwarden.com/contact/)。
{% endhint %}

如果账单在到期日之前未支付，您的账户以及适用的组织成员可能会受到影响。请支付账单以[恢复您组织的服务](organization-renewal.md#re-enabling-a-disabled-organization)或您的高级版方案。
